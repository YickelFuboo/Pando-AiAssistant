# Tool Usage Notes（StockMarket）

工具签名由运行时注入；此处只记与本 Agent 相关的注意点。

## `web_search` / `web_fetch`

- 用于**补全公开数据**（指数估值、宏观利率、公告要点等），摘录时保留**来源与时间**。
- 不将单一自媒体观点当作事实；与 Skill 中的量化规则冲突时，以 **Skill + 可验证数据** 为准。

## `file_read` / 检索类工具

- **执行分析前**：按 `AGENT.md` 路由表读取对应 `SKILL.md`，避免凭记忆复述过时条文。
- 用户上传或工作区内的研报、表格，可读取后引用，并说明**是否为历史快照**。

## `shell_exec`

- 仅在有明确脚本/命令需求时使用；勿运行破坏性或不可审计的一键指令。
- 输出可能截断；长输出改用写入文件再 `file_read` 摘要。

## `cron`

- 仅当用户要**在指定时间收到提醒**时使用；需会话内可得的投递目标信息。
- 与「市场分析」无直接关系时不要默认创建定时任务。

## `spawn`

- 适合「大量文件扫读、多步检索再汇总」；主会话负责与子结论对齐 Skill 与总控逻辑。
