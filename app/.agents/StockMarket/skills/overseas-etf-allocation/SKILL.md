# 海外市场ETF配置策略 Skill（极端高估期替代通道 · 地域分散终版）

## 角色定位
在A股和港股**同步进入极端高估**时激活的替代性配置通道。通过QDII基金将部分被迫闲置的现金配置到估值合理的海外市场宽基ETF中，实现地域分散、估值套利和资金利用率最大化。**本策略不是常规配置，仅在特定市场条件下激活。**

**核心约束：只从总控 Skill 划定的现金池中划拨资金，不扩大总仓位上限，不侵占A股/港股四大方向的正常配置额度。**

**本策略定位**：中长期地域分散配置，持股周期6-24个月，用简单宽基ETF实现跨市场暴露。
**资金约束**：资金从总控 Skill 的“现金/短债”池中临时划拨，上限为 min(现金池×40%, 总资产×15%)，确保无论总仓位如何变化，海外配置不超过总资产15%。
**退出条件**：A股或港股任一方退出极端高估→逐步清仓回国；海外市场自身估值进入高估→清仓。

## 触发条件
- 总控 Skill 判定A股（全指PE>95%）+港股（恒生PE>95%）同步进入极端高估
- 总仓位处于30%-50%低位，现金池占比40%-55%
- 用户请求“海外配置”、“QDII机会”、“全球分散”、“现金怎么增值”
- **非触发条件下不执行本策略，资金保留为现金管理Skill的短债/货基**

- 港股市场阶段（由总控 Skill 传入）：必须同步处于极端高估
- 现金池可用金额（由总控 Skill 传入）

- 各经济体十年期国债收益率
- QDII基金规模、日均成交、折溢价率、管理费、跟踪误差
- 人民币兑美元/欧元/日元/港元的汇率
- **当前日期和最近一个交易日的日期（强制获取，用于数据时效性标注）**

## 执行步骤

### 第一步：触发条件校验
1. **数据时效性强制校验**：
   - 明确标注当前分析日期和数据截止日
   - 若估值数据距今超过5个交易日，必须标注“⚠️ 数据已过期，结论可能失效”
2. **A股+港股同步极端高估确认**：
   - A股全指PE分位 > 95%
   - 港股恒生PE分位 > 95%
   - 若任一条件不满足，直接返回“当前不满足海外配置触发条件，资金保留为现金管理”，不执行后续步骤
3. **现金池确认**：从总控 Skill 获取当前现金池总额，海外配置上限 = min(现金池 × 40%, 总资产 × 15%)。此自适应规则确保无论总仓位处于30%下限还是50%上限，海外配置均不突破总资产的15%硬约束

### 第五步：常识校验与风险揭示
1. **核心目的重申**：本策略的初衷是解决极端高估期大量闲置资金仅获取货币收益（~2.5%）的机会成本问题。通过将部分闲余资金投向估值合理的海外市场，力争在承担适度风险的同时获取更高回报。
2. **风险核查**：
   - **汇率风险**：QDII基金以人民币计价，但底层资产是外币。若人民币大幅升值，即使是美股上涨，人民币净值也可能下跌。投资者需明确这是本策略的固有风险。
   - **流动性风险**：QDII基金规模普遍较小，日均成交偏低，大额申赎可能冲击净值。赎回周期长达T+3至T+5，需提前规划资金需求。
   - **溢价风险**：跨境ETF易出现折溢价异常波动，在恐慌或过热时溢价率可能冲高，建仓时溢价>3%暂停买入。
   - **国别风险**：海外市场面临如美联储意外加息、地缘政治冲突、新兴市场政策不稳等风险。
3. **反面论证**：
   - 如果A股仅高估3个月就回落，而海外配置的建仓成本（申赎费+汇兑损失）还未覆盖，本策略可能小幅亏损。
   - 如果人民币出现趋势性升值，海外配置的人民币回报可能显著低于该市场的本币回报。
   - **应对预案**：若上述情况发生，严格执行退出规则，不恋战、不追加。

## 错误处理
- 若某QDII基金折溢价率>3%，跳过该品种并注明“溢价过高，等回归后重新评估”。
- 若海外市场PE分位数据缺失，跳过该市场并注明“数据缺失”。
- 若触发条件不满足（A股或港股未同步极端高估），直接返回“当前不满足海外配置触发条件”。
- 若现金池不足等值5万人民币，返回“现金池过小，海外配置效率低，建议保持现金管理策略”。

## 备注
- **本策略非必选模块**，仅在特定市场环境下激活。用户可根据自身情况决定是否启用。
- 海外配置资金从总控 Skill 的现金池中划拨，不扩大总仓位上限。
- 海外配置上限 = min(现金池×40%, 总资产×15%)，自适应总仓位变化。
- 退出机制与总控 Skill 联动：A股/港股退出极端高估→海外仓位回归现金池。
- QDII基金存在汇率风险、高溢价风险和高费率特征，收益率需扣除申赎费（通常0.5%-1.5%）和汇兑损益。
- QDII基金赎回到账通常需T+3至T+5，退出计划需提前预留流动性。
- 本策略不追求深度穿透，不涉及个股选择，用宽基指数实现简单的地域分散。
- 本策略不覆盖美股个股、REITs或衍生品。
