# 海外市场ETF配置策略 Skill（全球价值捕手 · 主动跨市场终版）

## 角色定位
这是一个主动型的全球价值捕手。它持续监控全球主要市场的估值变化，通过比较各国市场的风险收益比，在总仓位的框架内，主动捕捉海外市场优于A股的阶段性配置机会。它不再是被动等待A股“出事”的替代品，而是从全球视角出发，为投资组合寻找最优风险收益比的一个主动工具。

**核心策略：全球比价，寻找洼地。通过比较海外市场与A股的估值分位差异，主动捕捉海外市场的相对价值。**

**本策略定位**：中长期地域分散配置，持股周期6-24个月，用简单宽基ETF实现跨市场暴露。
**资金约束**：资金来源于总控 Skill 划定的现金池或价值龙头闲置额度，**硬性上限为总资产的15%**。
**退出条件**：海外市场自身估值进入高估，或其相对A股的估值优势消失。
**持续跟踪**：每周扫描全球估值变化，对已持仓标的进行持仓诊断并刷新止盈止损信号。

## 触发条件（双轨制）
本策略有两条独立的启动路径，满足任一即可激活：

**路径一：A股泡沫避险**
- A股全指PE分位 > 95% + 港股恒生PE分位 > 95%（同步极端高估）
- 资金从现金池划拨，上限 = min(现金池×40%, 总资产×15%)

**路径二：全球价值捕捉**
- 目标海外市场PE分位 < 30%（处于合理低估或深度低估区间）
- 且该市场PE分位**低于沪深300 PE分位20个百分点以上**（相对价值显著优于A股）
- 资金优先从价值龙头类闲置额度中划拨，若不足再从现金池划拨，上限 = 总资产 × 10%
- **A股处于系统性崩盘阶段时，暂停路径二，优先保障本土资金安全**

> **路径二的优先级逻辑**：价值龙头类资金本身追求中长期回报，与海外配置的6-24个月持有周期匹配。优先使用价值龙头类的闲置额度，可以保持现金池的流动性储备不受影响。

## 执行步骤

### 第一步：前置参数与运行模式判定
1. **数据时效性强制校验**：明确标注当前分析日期和数据截止日。若估值数据距今超过 5 个交易日，必须标注“⚠️ 数据已过期”。
2. **运行模式判定**：
   - 若A股+港股同步极端高估 → 执行路径一，资金从现金池划拨
   - 若发现目标市场的PE分位 < 30%，且低于沪深300 PE分位20pp以上 → 执行路径二
   - 若两条路径同时满足 → 优先执行路径一
   - 若两条路径都不满足 → 返回“当前无符合条件的海外配置机会”
3. **A股系统性崩盘检查**：若A股处于系统性崩盘阶段，路径二自动暂停。
4. **存量持仓检查**：若用户已持有海外QDII，先执行第二步持仓诊断，再进入新机会扫描。

#### 4.3 建仓规则
- 全部采用分批建仓，每批次间隔至少3个交易日
- 分3批建仓，每次建1/3目标仓位
- 单次建仓金额 ≥ 等值2万人民币
- **溢价率>3%时暂停该品种建仓**，等折溢价回归正常
- **建仓前必须核查该QDII是否处于暂停申购状态**：
  - 若**正常申购**：按计划分批买入
  - 若**暂停申购**但二级市场**溢价<2%**：可在场内买入，并标注“暂停申购，通过二级市场溢价买入”
  - 若**暂停申购**且二级市场**溢价>2%**：放弃该品种，注明“因暂停申购且溢价过高，无法建仓”，资金暂留现金池或切换到替代品种
- 对流动性标注为“⚠️分批”的标的，单次建仓金额不超过5万元，若目标仓位超过5万，自动增加分批次数

#### 4.4 止盈与止损路径（每只标的独立执行）

**信号优先级规则**：若多条止盈/止损规则同时触发，**按动作幅度最大的规则执行，不叠加**。例如同时触发“PE>50%减1/3”和“折价<10pp减1/2”，执行减1/2。触发后其他信号在下次周度扫描时重新评估。

| 触发条件 | 动作 | 适用场景 | 信号类型 |
| :--- | :--- | :--- | :--- |
| PE分位升至50% | 减仓1/3 | 自身估值进入合理偏高区间 | 止盈 |
| PE分位升至60% | 再减仓1/3 | 自身估值进入高估区间 | 止盈 |
| PE分位升至70% | 清仓 | 自身估值泡沫化 | 止盈 |
| 相对A股折价收窄至<10pp | 减仓1/2 | 海外市场的相对优势减弱 | 止盈 |
| 相对A股折价消失（PE分位>沪深300PE分位） | 清仓 | 海外不再有估值优势 | 止盈 |
| A股退出极端高估（仅路径一触发时适用） | 分4周逐步清仓 | 避险需求解除 | 退出 |
| 人民币单月升值 > 3% | 减仓50%，重新评估 | 汇率损失抵消海外收益 | 止损 |

### 第六步：常识校验与风险揭示
1. **核心目的重申**：本策略通过全球比价，寻找风险收益比优于A股的海外市场机会。它不是单纯的分散投资，而是主动的价值捕捉。
2. **风险核查**：
   - **汇率风险**：QDII基金以人民币计价，人民币升值会侵蚀海外收益
   - **流动性风险**：QDII基金规模较小，赎回周期T+3至T+5。部分标的（513030、159612）规模偏小，单次建仓超5万需分批多日执行
   - **溢价风险**：建仓时溢价>3%暂停；持仓中溢价>5%卖出1/2
   - **外汇额度风险**：QDII基金可能因外汇额度用完而暂停申购。建仓前必须核查申购状态——若暂停且溢价>2%，放弃该品种
   - **国别风险**：海外市场面临美联储政策、地缘政治等特有风险
3. **反面论证**：
   - 如果海外市场低估值持续数年不修复，本策略可能长期低效
   - 如果人民币出现趋势性升值，海外配置的人民币回报可能大幅缩水
   - **应对预案**：严格执行相对折价消失的清仓规则，不恋战

## 错误处理
- 若某QDII基金折溢价率>3%，跳过该品种并注明“溢价过高”。
- 若某QDII基金暂停申购且二级市场溢价>2%，注明“因暂停申购且溢价过高，无法建仓”。
- 若海外市场PE分位数据缺失，跳过该市场并注明“数据缺失”。
- 若现金池不足等值5万人民币且价值龙头类无闲置额度，返回“资金不足”。
- 若两条路径均不满足，返回“当前无符合条件的海外配置机会”，并列出各市场需等待的量化条件。
- 若A股处于系统性崩盘阶段，路径二自动暂停，仅路径一可执行。
- 对流动性偏弱的标的，若目标仓位超过5万，自动增加分批次数并提示。

## 备注
- 海外总仓位硬性上限为总资产的15%
- 同一跟踪指数的多只QDII，优选规模和流动性最好的品种
- QDII赎回到账通常需T+3至T+5，退出计划需预留流动性
- 本策略不涉及个股选择和衍生品
- 相对价值比较基准默认为沪深300 PE分位
- 外汇额度风险是QDII特有风险，建仓前必须核查申购状态
- 路径二优先使用价值龙头闲置额度，以保护现金池流动性
- 多条止盈/止损信号同时触发时，按动作幅度最大的规则执行，不叠加
