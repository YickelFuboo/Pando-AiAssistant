# 海外市场ETF配置策略 Skill（全球价值捕手 · 主动跨市场终版）

## 角色定位
这是一个主动型的全球价值捕手。它持续监控全球主要市场的估值变化，通过比较各国市场的风险收益比，在总仓位的框架内，主动捕捉海外市场优于A股的阶段性配置机会。它不再是被动等待A股“出事”的替代品，而是从全球视角出发，为投资组合寻找最优风险收益比的一个主动工具。

**核心策略：全球比价，寻找洼地。不仅看估值便宜，还看增长前景——低估值但负增长的“价值陷阱”必须回避。增长前景直接影响建仓节奏和止盈灵敏度。**

**本策略定位**：中长期地域分散配置，持股周期6-24个月，用简单宽基ETF实现跨市场暴露。
**资金约束**：资金来源于总控 Skill 划定的现金池或价值龙头闲置额度，**硬性上限为总资产的15%**。
**退出条件**：海外市场自身估值进入高估，或其相对A股的估值优势消失。
**持续跟踪**：每周扫描全球估值变化，对已持仓标的进行持仓诊断并刷新止盈止损信号。

## 触发条件（双轨制）
本策略有两条独立的启动路径，满足任一即可激活：

**路径一：A股泡沫避险**
- A股全指PE分位 > 95% + 港股恒生PE分位 > 95%（同步极端高估）
- 资金从现金池划拨，上限 = min(现金池×40%, 总资产×15%)

**路径二：全球价值捕捉**
- 目标海外市场PE分位 < 30%（处于合理低估或深度低估区间）
- 且该市场PE分位**低于沪深300 PE分位20个百分点以上**（相对价值显著优于A股）
- 资金优先从价值龙头类闲置额度中划拨，若不足再从现金池划拨，上限 = 总资产 × 10%
- **A股处于系统性崩盘阶段时，暂停路径二，优先保障本土资金安全**

## 策略运作逻辑（通俗版）

**在哪些情况下，这个策略大概率能赚到钱？**
- **全球温和增长 + A股高估时**：海外低估值市场吸引全球资金回流，估值修复推动ETF净值上涨。我们赚的是“从低估回到合理”的钱。
- **全球同步复苏时**：海外市场的盈利增长（EPS提升）推动指数上涨，即使估值不修复，也能赚到盈利增长的钱。
- **人民币贬值时**：海外资产以外币计价，换算回人民币会产生额外汇兑收益，增强整体回报。

**在哪些情况下，这个策略可能会亏钱或跑输？**
- **中国一枝独秀时**：A股走出独立牛市，我们部分资金在海外低估值市场，会显著跑输A股。这时要理解：我们放弃的是“追涨A股”的机会，换取的是“A股崩盘时的保护”。
- **人民币持续升值时**：海外资产以外币计价，换算回人民币会产生汇兑损失，侵蚀海外收益。这是本策略的固有成本。
- **海外低估值长期不修复时**：如果海外市场因结构性原因（如人口老龄化、技术落后）长期处于低估值状态，我们的资金可能长期低效。这时需要重新评估该市场的增长预期。

**为什么要坚持这个策略？**
- 它的核心价值不在于“任何时候都跑赢”，而在于“在A股最危险的时候提供保护”。当A股泡沫破裂时，海外低估值市场通常跌幅更小，我们的组合整体回撤更可控。
- 它让我们在全球范围内寻找性价比最优的资产，而不是把鸡蛋都放在一个篮子里。

## 执行步骤

### 第一步：前置参数与运行模式判定
1. **数据时效性强制校验**：明确标注当前分析日期和数据截止日。若估值数据距今超过 5 个交易日，必须标注“⚠️ 数据已过期”。
2. **运行模式判定**：
   - 若A股+港股同步极端高估 → 执行路径一，资金从现金池划拨
   - 若发现目标市场的PE分位 < 30%，且低于沪深300 PE分位20pp以上 → 执行路径二
   - 若两条路径同时满足 → 优先执行路径一
   - 若两条路径都不满足 → 返回“当前无符合条件的海外配置机会”
3. **A股系统性崩盘检查**：若A股处于系统性崩盘阶段，路径二自动暂停。
4. **存量持仓检查**：若用户已持有海外QDII，先执行第二步持仓诊断，再进入新机会扫描。

**极端行情应急预案**：若持仓品种跟踪的指数单日跌幅超过5%，且当日收盘后相对沪深300的折价急剧拉大超过10个百分点，说明市场出现恐慌性错杀。此时可以提前执行下一批次建仓（跳过回调等待），在次日开盘直接买入。

### 第六步：常识校验与风险揭示
1. **核心目的重申**：通过全球比价，寻找风险收益比优于A股的海外市场机会。不仅看估值便宜，更看增长前景——增长前景直接影响建仓节奏和止盈灵敏度。
2. **风险核查**：汇率风险、流动性风险、溢价风险、外汇额度风险、增长预期风险（价值陷阱）、国别风险。
3. **反面论证**：
   - 海外低估值长期不修复→可能长期低效
   - 人民币趋势性升值→汇兑损失侵蚀收益
   - EPS增速持续下调→低估值可能是基本面恶化的反映而非机会
4. **策略表现预期管理**：
   - 当A股泡沫破裂时，海外低估值市场通常跌幅更小，组合整体回撤更可控
   - 当全球同步复苏时，海外市场因估值更低，向上弹性可能更大
   - 当中国一枝独秀时，本策略会显著跑输A股，这是分散投资的固有代价

- 折溢价率>3%：跳过并注明。

- PE分位数据缺失：跳过并注明。
- EPS增速预期数据缺失：跳过增长评估并注明，默认为停滞增长。
- 现金池不足等值5万且价值龙头无闲置：返回“资金不足”。
