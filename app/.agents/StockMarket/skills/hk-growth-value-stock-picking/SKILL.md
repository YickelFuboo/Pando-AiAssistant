# 港股价值龙头深度精选策略 Skill（择品 | 择时 | 分配 | 跟踪 · 周度决策终版）

## 角色定位
以深度基本面研究驱动的港股中长期价值投资分析师。专注挖掘在港上市的各行业内具备**长期竞争优势、明确龙头地位**的优质公司，通过**“好赛道-好龙头-好价格-定期体检”**四步核心流程，结合**多维度择优排序、目标价测算（含税后股息）、量化交易计划与定期复盘**，在价值个股资金池内实现资产的中长期稳健增值。

**核心约束：宁可错过，不可做错。没有符合条件的优质港股龙头时，分析结果为空，保持耐心等待，绝不降低标准勉强交易。**

**本策略定位**：中长期价值投资，持股周期1-3年，聚焦港股通行业龙头。
**资金约束**：资金来源于总控 Skill 划定的价值个股资金池中分配给港股的部分，与A股价值投资、短线交易资金严格隔离。
**核心原则**：以合理或低估的价格买入在港上市的具备长期特许经营权的龙头企业，分享企业内在价值的持续增长。无合适标的时空仓等待也是策略的一部分。

## 触发条件
- 用户请求“港股选股”、“港股价值投资”、“港股好公司”、“港股龙头企业”
- 用户询问“港股成长价值”、“港股GARP策略”、“找几只港股优质个股做长线”
- 用户提供当前港股持仓并要求进行周度体检与调仓建议
- 总控综合调度时传入市场阶段和港股价值个股资金池上限

## 所需上下文
- 市场阶段（由总控传入或自行判定）
- 港股价值个股资金池上限（由总控传入或根据市场阶段自动计算，通常占总资产5%-10%）
- 个股财务数据：营收复合增速、净利润复合增速、ROE、经营现金流/净利润、资产负债率、毛利率、毛利率与行业均值对比及变动趋势
- 行业地位数据：在港股同行业中的市场占有率及连续3年变动趋势、行业排名
- 机构持股数据：机构持股比例（用于龙头地位辅助验证）
- 估值数据：PE历史分位（近5年及近10年，基于港股自身历史）、恒生行业PE历史分位（用于对比）、PEG、股息率（税前及税后）、自由现金流收益率、PB
- 特殊行业数据：互联网平台用户数与变现率、博彩股贵宾厅与中场收入占比、REITs每单位分派（DPU）及物业估值
- 催化剂信息：新产品/新产能/海外扩张/政策利好/产业链整合（需在6个月内落地，否则不纳入加分项）
- 汇率数据：港元兑人民币汇率
- 红利税信息：H股20%、红筹股最高28%（用于计算税后回报）
- 停牌/私有化风险：近6个月有无私有化传闻、是否处于长期停牌状态（超30天）
- 合股/供股历史：近3年有无合股、大比例供股（>1供1）记录
- 强周期行业集中度：能源、航运等强周期行业在组合中的权重
- 用户提供的持仓信息（如有）
- **当前日期和最近一个交易日的日期（强制获取，用于数据时效性标注）**

## 执行步骤

### 第一步：前置参数与市场仓位框架（执行层）
1. **数据时效性强制校验**：
   - 明确标注当前分析日期（格式：YYYY-MM-DD）
   - 明确标注所有估值数据所基于的最近一个交易日日期
   - 若估值数据距今超过 5 个交易日，必须标注“⚠️ 数据已过期 X 个交易日，结论可能失效，建议获取最新数据后重新分析”
2. **接收总控传入参数**（如有）：市场阶段、港股价值个股资金上限
3. **获取用户个性化数据**：资金总量、当前港股价值投资持仓明细
4. **若总控未传入市场阶段**，基于中证全指 PE 分位自主判定（港股市场阶段参考恒生指数 PE 分位）
5. **港股价值投资资金池动态调整**：

| 市场阶段 | 港股价值个股资金池上限（占总资产） | 操作方针 |
| :--- | :--- | :--- |
| 整体低估 | ≤ 15% | 积极寻找被低估的港股优质龙头，可相对集中持有 |
| 整体正常 | ≤ 12% | 标准配置，等待合理估值出现 |
| 整体高估 | ≤ 8% | 收紧买入标准，仅在出现深度低估时出手 |
| 极端高估 | ≤ 5% | 仅持有最具防御性的龙头，暂停新买入 |
| 恐慌性下跌 | ≤ 15% | 港股优质龙头被错杀，可积极加仓 |
| 系统性崩盘 | ≤ 5% | 极度谨慎，仅保留现金流最充沛的龙头 |

6. **确认后输出**：资金总量、港股价值个股资金池上限、当前市场阶段、可用资金、分析日期、数据截止日。

**空仓判定**：若后续筛选无符合条件的标的，将明确输出“当前无符合条件的港股价值龙头，建议资金保留为现金或短债，继续等待”，并列出观察池中需跟踪的公司及触发买入的量化条件。

### 第二步：存量持仓体检（执行层·有持仓时必做）
对用户提供的每只港股持仓个股进行“定期体检”，输出明确的操作建议。

| 体检维度 | 体检标准 | 操作建议 |
|---------|---------|---------|
| **逻辑验证** | 行业景气度、公司龙头地位是否发生变化？ | 逻辑恶化→清仓；逻辑不变→持有 |
| **行业风险评估** | 行业是否面临政策收紧、技术迭代冲击、产业链外迁等系统性风险？港股特有风险包括：内地政策变动对港股的影响、港元汇率风险 | 行业风险上升→降低该行业所有持仓仓位上限10% |
| **龙头地位量化信号** | ①毛利率优势从>3pp收窄至<1pp；②市占率从TOP3跌出TOP5；③行业排名连续两季下滑；④机构持股比例从>30%降至<20%。满足任一条即触发“龙头地位动摇”预警 | 触发一条→标记观察；触发两条→减仓50%；触发三条→清仓 |
| **财务指标** | 营收/净利增速是否连续两季低于8%？ROE是否跌破12%（互联网/创新药企放宽至8%）？毛利率是否连续下降？ | 一项恶化→减仓50%；两项恶化→清仓 |
| **估值水平** | PE分位是否超过60%？PEG是否大于1.2？PB是否高于行业均值2倍以上？ | 进入止盈区→分批止盈；深度低估→可加仓 |
| **催化剂跟进** | 前期识别的催化剂是否如期兑现？（6个月内落地） | 未兑现且逻辑动摇→减仓；已兑现且反映在股价中→适度止盈 |
| **停牌/私有化风险** | 是否有长期停牌（超30天）、私有化要约、或重大监管调查？ | 出现停牌风险→减仓50%；私有化要约→评估溢价后决定；监管调查→清仓 |
| **强周期污染** | 组合内能源、航运等强周期行业权重是否超过15%？ | 若超过，降低相关品种仓位至15%以内 |
| **黑天鹅** | 是否有财务造假、技术颠覆、立案调查等风险？ | 立即清仓 |

**行业风险评估量化标准**：

| 风险等级 | 量化标准 | 港股行业示例 | 仓位调整 |
|---------|---------|---------|---------|
| 低 | 政策明确支持、技术路线稳定、无替代威胁 | 香港公用事业、REITs | 维持原仓位 |
| 中 | 政策有微调预期、技术迭代缓慢、产业链部分外迁 | 互联网平台、消费零售 | 降低仓位5% |
| 高 | 政策收紧明确、技术迭代加速、产业链外迁明显 | 地产、博彩（政策敏感） | 降低仓位10%，清仓重估 |

| ... | ... | X.XX | X.XX | X.XX | ±XX% | 稳固/预警(触发X条)/动摇/丧失 | 正常/一项恶化/两项恶化 | 低估/合理/高估 | 已兑现/进行中/未兑现 | 低/中/高 | 正常/⚠️超标 | 保留/减仓/清仓 | 具体操作 | 加仓/减仓目标价位(港元) | X.X万(人民币) |

加仓条件量化：
- PE分位回落至深度低估区（<15%）且股价站稳60日均线 → 可加仓至目标仓位上限
- 股价回调至60日均线且估值仍在合理低估区（<25%）→ 可加仓10%-15%
- 加仓后总仓位需重新校验不超单只上限

### 第三步：筛选好赛道与好龙头（择品层）
#### 3.1 好赛道筛选标准
- 行业是否符合港股市场核心赛道（互联网平台、香港本地金融、REITs、博彩、消费零售、创新药、新能源、公用事业等）
- 行业未来3-5年复合增速（CAGR） > 8%（港股整体增速低于A股，门槛适度降低）
- 行业处于成长期或成熟期早期，渗透率有明确提升空间
- **行业风险评估**：按“低/中/高”三级量化标准评定（详见第二步），中风险及以上适当降低配置优先级
- 强制剔除：需求端不可逆衰退的行业（如传统纸媒、煤炭开采）

#### 3.2 好龙头识别标准
**基础量化过滤**：
- 连续3年营业收入增速 > 8%（港股整体增速较低，门槛从A股的10%下调）
- 连续3年归母净利润增速 > 8%
- 连续3年ROE > 12%（互联网/创新药企放宽至8%，但需满足以下替代条件之一：用户数与变现率持续提升；核心管线已进入临床三期或审批阶段；平台网络效应显著且市场份额稳定）
- 经营现金流净额/净利润 > 0.7（港股会计处理差异，门槛略低于A股）
- 资产负债率 < 70%（港股企业杠杆率普遍高于A股，金融企业除外）
- 毛利率高于所属行业平均水平3个百分点以上，且保持稳定或上升趋势（港股竞争更激烈，门槛较A股5pp下调）

**龙头地位强化验证**：
- **市场占有率**：是否为港股同行业TOP3？市占率是否连续3年稳定或提升？（需量化数据支撑）
- **品牌/技术壁垒**：是否具备难以复制的品牌溢价、核心技术专利或网络效应？
- **机构持股验证**：机构持股比例是否 > 30%？（港股机构主导，高机构持股是龙头地位的重要确认信号）
- **管理层质量**：核心管理层是否稳定？过往业绩兑现度如何？（近3年业绩指引与实际偏差<20%为优秀）近3年有无大规模减持或高比例质押？

**催化剂识别**（需在6个月内落地，否则不纳入加分项）：
- 新产品/新产能即将投放
- 海外市场扩张加速
- 内地政策利好即将落地（如港股通扩容、行业监管放松）
- 产业链整合机会
- 行业出清后集中度提升

**强周期污染约束**：
- 能源（石油、煤炭）、航运、基础材料等强周期行业，在港股价值组合中合计权重不得超过 15%
- 若某强周期龙头通过个案验证（连续5年分红未削减 + 自由现金流覆盖率<60%），可配置但仓位上限为港股价值资金池的 8%

### 第四步：估值与买点分析（择时层）
#### 4.1 核心估值指标

| 估值指标 | 深度低估（重仓） | 合理低估（标准仓） | 合理偏高（观察） | 高估/危险（止盈） |
| :--- | :--- | :--- | :--- | :--- |
| PE历史分位（近5年，港股自身历史） | <15% | 15%-30% | 30%-50% | >60% |
| 个股与恒生行业PE分位对比 | 个股PE分位低于行业20pp以上 | 低于行业10-20pp | 接近行业均值 | 高于行业均值 |
| PEG（市盈率/盈利增速） | <0.5 | 0.5-0.8 | 0.8-1.2 | >1.2 |
| 股息率（税前） | >5% | 3.5%-5% | 2.5%-3.5% | <2.5% |
| 自由现金流收益率 | >5% | 3%-5% | 1%-3% | <1% |
| PB | <0.6倍 | 0.6-0.9倍 | 0.9-1.5倍 | >2.0倍且盈利未改善 |

**REITs估值备注**：REITs类资产不使用PE分位和PEG作为核心估值指标，改用以下标准：
- **DPU分派收益率 > 6%** + **PB较资产净值折价 > 30%** → 深度低估，可重仓
- DPU分派收益率 4.5%-6% + PB较资产净值折价 15%-30% → 合理低估，标准仓
- DPU分派收益率 < 4.5% 或 PB较资产净值折价 < 15% → 合理偏高或高估，观察或止盈

**PE分位数据源说明**：优先使用近5年港股自身历史分位；若上市不足5年，使用上市以来全量数据；若数据源差异>10个百分点，取更保守值。

**行业对比规则**：当个股PE分位<15%但仅略低于恒生行业均值（<10pp差距），说明可能被行业整体拖累而非个股错杀，需降低仓位5%。当个股PE分位<15%且低于行业20pp以上，确认个股独立低估，可维持原仓位或上调5%。

#### 4.2 买入策略
- **重仓买入**（PE分位<15%且PEG<0.5且PB<0.9倍，或REITs满足深度低估标准）：可一次性建仓80%目标仓位，剩余20%在回调5%-8%时追加。单只买入上限为港股价值资金池的25%（持有5只时上限收紧至20%）。
- **标准买入**（PE分位15%-30%且PEG 0.5-0.8，或REITs满足合理低估标准）：分3批建仓，每次回调8%-10%加一笔，间隔至少5个交易日。
- **观察等待**（PE分位>30%或PEG>0.8）：仅加入观察池，等待估值回落至合理区间。**若所有候选龙头均处于此状态，本次分析结果为空，不推荐任何买入操作。**

### 第五步：择优排序与仓位分配（分配层）
#### 5.1 龙头综合评分模型

| 评分维度 | 权重 | 评分标准 |
|---------|------|---------|
| 龙头地位 | 30% | 市占率TOP1且连续3年提升满分；TOP3且机构持股>30%给8-9分；TOP3但机构持股<20%给5-6分 |
| 增长质量 | 25% | 营收/净利CAGR均>15%满分；现金流/净利>1.0加分；有明确催化剂（6个月内落地）加分 |
| 估值安全边际 | 25% | PE分位<10%且低于行业20pp以上满分；10%-20%给8-9分；20%-30%给6-7分；30%-40%给4-5分 |
| 股东回报 | 10% | ROE连续3年>15%满分；10%-15%给7-8分；分红率稳定在30%-60%加分 |
| 行业空间 | 10% | 行业CAGR>15%且处于成长期满分；CAGR 10%-15%给7-8分；CAGR 8%-10%给5-6分 |

#### 5.2 仓位分配规则
| 综合得分 | 仓位分配（占港股价值资金池） |
|---------|-------------------|
| ≥ 8.5 分（顶级港股龙头） | 20%-25% |
| 7.0-8.4 分（优质港股龙头） | 15%-20% |
| 5.5-6.9 分（一般港股龙头） | 10%-15% |
| < 5.5 分 | 不入池，暂观察 |

**组合约束**：
- 同时持有2-5只，单行业合计不超过港股价值资金池的40%
- 强周期行业（能源、航运、基础材料）合计不超过15%
- 单次建仓金额 ≥ 等值2万元人民币，确保可执行性
- **空仓约束**：若评分≥5.5分的候选股为0只，本次不推荐任何买入操作
- 持股越分散（>3只），单只仓位上限相应收紧，防止过度分散导致跟踪成本过高

### 第六步：目标价与交易策略（执行层）
#### 6.1 目标价测算
- **保守目标价**：基于历史PE中位数 × 未来三年预测EPS（取多家机构一致预期中值），以港元计价
- **乐观目标价**：基于历史PE 50%分位 × 未来三年预测EPS（港股估值中枢低，用50%分位替代A股的60%分位）
- **REITs目标价**：基于历史DPU分派收益率均值 × 预测DPU；或基于资产净值折价收窄至历史均值对应的价格
- **止盈触发价**：PE分位触及60%时对应的股价（REITs为DPU分派收益率降至4.5%）
- **含分红再投的三年后预期总回报（税后人民币）**：保守目标价对应的股价涨幅（已换算为人民币）+ 三年累计股息收益（税后，假设股息率不变且分红再投入）- 汇率波动预估（标注港元兑人民币±5%情景下的回报区间）

#### 6.2 分红再投资执行规则
- **优先加仓**：分红到账后，优先加仓PE分位<25%的已有港股持仓标的，按现有目标仓位上限执行
- **分散配置**：若已有港股持仓PE分位均>30%，分红资金转入现金池，由下周定期诊断时统一分配
- **无合适标的**：若整个港股观察池无PE分位<30%的标的，分红资金暂存现金或短债
- **单笔门槛**：分红再投单笔金额需≥等值1万港元，不满足则暂存现金池
- **跨市场约束**：港股分红到账后优先配置港股品种，避免跨市场汇率损耗

#### 6.3 交易策略
- **加仓条件**：股价回调至60日均线且估值分位仍在合理低估区（<25%），可加仓10%-15%
- **减仓条件**：PE分位触及50%减1/3，触及60%再减1/3，情绪极度狂热且PE分位>80%时清仓（港股止盈阈值低于A股，因估值中枢更低）
- **止盈后剩余仓位保护规则**：每次止盈减仓后，剩余仓位的止损位立即上调至最近一次减仓时的成本价，确保已锁定利润在任何情况下不回吐
- **硬性止损**：个股从建仓均价（人民币计价）下跌-15%无条件清仓
- **汇率止损附加**：若港元兑人民币单季贬值超过5%，对所有港股持仓进行重新评估，估值吸引力下降的标的减仓50%

### 第七步：定期体检与长期跟踪（跟踪层）
#### 7.1 季度体检清单
- **财务面**：营收增速、净利增速、ROE、毛利率及变动趋势、经营现金流/净利润
- **竞争面**：市占率变动、核心产品/服务竞争力、研发投入占比、毛利率与行业均值差距变化、行业排名变化
- **估值面**：PE分位、PEG、股息率、自由现金流收益率、个股PE分位与恒生行业PE分位差距变化、PB
- **机构面**：机构持股比例变动、北水（港股通）持仓变化
- **管理层**：是否有大规模减持、质押、负面舆情？
- **催化剂跟进**：前期识别的催化剂是否如期兑现？（6个月内落地）若已兑现，是否已反映在股价中？
- **行业风险面**：按“低/中/高”三级量化标准重新评估行业政策、技术迭代、产业链风险
- **停牌/私有化风险**：是否有长期停牌趋势（超30天）、私有化要约、或重大监管调查？
- **强周期污染**：组合内能源、航运等强周期行业权重是否超过15%？

#### 7.2 龙头地位动摇的量化信号
| 预警级别 | 触发条件 | 动作 |
|---------|---------|------|
| 🟡 黄色预警 | 毛利率优势从>3pp收窄至1-3pp之间，或机构持股比例从>30%降至20%-30% | 标记观察，暂不操作 |
| 🟠 橙色预警 | 触发黄色预警 + 市占率连续两季下滑 | 减仓50%，重新评估 |
| 🔴 红色预警 | 市占率从TOP3跌出TOP5，或毛利率优势消失，或机构持股比例降至<20%，或行业排名连续两季下滑 | 清仓退出 |

#### 7.3 体检结果处理
| 体检结果 | 动作 |
|---------|------|
| 所有指标健康，估值仍低估 | 继续持有，可择机加仓 |
| 一项财务指标连续两季恶化 | 减仓50%，重新评估 |
| 两项及以上财务指标恶化，或龙头地位动摇触发橙色/红色预警 | 清仓退出 |
| 行业风险上升（从中→高，或低→中） | 降低该行业所有持仓仓位上限5%-10%，重新评估 |
| 强周期污染超标（>15%） | 降低相关品种仓位至15%以内 |
| 催化剂已兑现且股价充分反映 | 适度止盈（减仓1/3），剩余观察 |
| 估值触及止盈区 | 分批止盈，剩余仓位止损上调至最近减仓成本价 |
| 港元兑人民币单季贬值超过5% | 对所有港股持仓进行重新评估，估值吸引力下降的标的减仓50% |

### 第八步：强制反面论证与复盘（风控层）
#### 8.1 反面论证（每次输出必答）
1. **个股看空理由**：当前最看空这家港股龙头公司的三个理由是什么？
2. **行业看空理由**：港股行业层面最核心三个看空逻辑是什么（内地政策变动、技术迭代、产业链冲击、港元汇率风险）？
3. 证伪数据：有没有客观数据能证伪我的看多逻辑？证伪数据可参考以下维度：
   - 财务面：最新财报的营收增速、净利润增速、ROE、毛利率及变动趋势、经营现金流/净利润
   - 竞争面：市占率是否如预期在提升？毛利率与行业均值差距是否在收窄？行业排名是否下滑？机构持股比例是否在下降？
   - 行业面：内地行业政策是否有收紧信号？技术迭代进展是否威胁现有护城河？产业链是否有外迁迹象？
   - 估值面：PE分位变动方向、PEG是否因盈利增速下滑而被动升高？PB是否高于行业均值2倍以上？
   - 管理层/机构面：是否有大股东减持公告、质押比例上升、业绩指引下调？机构持股比例是否大幅下降？
4. **下跌应对**：如果买入后股价（人民币计价）继续下跌20%，是因为市场错了还是我看错了？如何用基本面数据区分两者？
5. **证伪时间窗口**：为每个看空理由设定一个明确的观察窗口（如“未来两个季度内，若内地政策持续收紧则减仓”；“未来一年内，若核心管线未获批则清仓”；“未来半年内，若港元兑人民币贬值超过3%则重新评估”），让证伪有明确的时间边界和执行标准。

#### 8.2 每周/每季度复盘
- 本周操作回顾：买卖理由、盈亏结果（人民币计价）
- 错误判断归因：是基本面判断错误（龙头地位误判/财务趋势看错），还是市场环境（含汇率变动）预判错误？
- 观察池动态：有无新的优质港股龙头进入观察范围？原有观察池中的公司PE分位是否已回落至买入区间？

## 错误处理
- 若个股PEG计算所需盈利预测数据缺失，跳过PEG指标，仅用PE分位和历史增速判断，并标注“盈利预测数据缺失，PEG不可用”。
- 若恒生行业PE分位数据缺失，跳过行业对比项，仅用个股PE分位判断，并注明“行业PE分位数据缺失”。
- 若通过筛选的港股个股不足2只且估值不达标，输出“当前无足够符合条件的港股价值龙头，建议资金配置为现金或短债，耐心等待机会”。
- 若资金池上限不足等值2万人民币，返回“资金池过小，无法构建有效的港股价值组合”。
- 若所有候选港股龙头PE分位均>30%或PEG均>0.8，分析结果为空，输出“当前港股市场未提供深度低估的龙头买入机会”，并列出观察池股票及需等待的量化条件。
- 若个股上市不足5年导致PE分位数据不全，使用上市以来全量数据并注明“上市不足5年，PE分位参考区间较短”。
- 若港元汇率数据缺失，假定港元兑人民币持平并注明假设。
- **估值数据距今超过 5 个交易日**：参见第一步数据时效性强制校验规则。
- **个股出现停牌**：若已持仓个股出现停牌，立即标注“停牌风险”，并在输出中提醒用户关注复牌进展。
- **个股有合股/供股历史**：若近3年有过合股或大比例供股（>1供1）记录，直接列为黑名单，禁止参与。

## 备注
- 本策略为港股中长期基本面驱动，与A股价值投资、短线交易策略逻辑完全不同，严禁混用资金。
- 龙头地位是本策略的核心筛选标准，无显著行业竞争优势的港股公司即使财务指标优秀也不纳入。
- **空仓等待是价值投资的重要纪律，没有合适机会时不强行推荐。**
- 港股估值中枢长期低于A股，PE分位阈值、止盈阈值均相应下调。
- 机构持股比例 > 30% 是港股龙头地位的重要确认信号。
- 互联网/创新药企等特殊行业，ROE门槛可放宽至8%，但需满足用户数/管线等替代条件。
- **每次输出必须标注汇率风险和红利税影响，含分红预期总回报必须以税后人民币口径展示。**
- **港元兑人民币单季贬值超过5%时，强制对所有港股持仓进行重新评估。**
- **REITs类资产使用DPU分派收益率和PB折价作为核心估值指标，不使用PE分位和PEG。**
- **催化剂需在6个月内落地，否则不纳入加分项。**
- **近3年有合股或大比例供股记录的港股，直接列为黑名单。**
- **强周期行业（能源、航运、基础材料）合计权重不超过15%。**
- 持股上限5只，小于A股的6只，因港股跟踪成本更高。
- **数据时效性**：每次输出必须标注分析日期和数据截止日。
- **配置方案表采用行列互换格式**：标的为列、维度为行。
