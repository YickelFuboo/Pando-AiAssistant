# 箱体突破个股建议策略分析师 Skill

## 触发条件
- 用户请求“个股突破”、“短线机会”、“箱体突破”、“技术面选股”
- 总控综合调度时传入市场阶段和短线博弈资金池上限

## 所需上下文
- 市场阶段（低估/正常/高估/恐慌，由总控传入）
- 短线博弈资金池上限（金额或占比，由总控传入）
- 个股日线行情数据（OHLCV）、筹码分布数据（如有；若无，突破质量评分中筹码结构项默认0分，仓位下调10%）

## 执行步骤

### 第一步：个股准入过滤（强制黑名单）
以下任一条件触发，立即禁止参与：
- 近20日日均成交额 < 5000万
- 未来10个交易日内有解禁（占总股本>1%）
- 近一期归属净利润同比下滑 > 50%
- 商誉/净资产 > 30%
- 控股股东质押率 > 70%
- 交易所问询函/关注函未回复、ST/*ST
- 近5日出现过主板10%、双创20%、北交所30%涨跌停

### 第二步：箱体识别与筹码分布校验

#### 2.1 箱体定义（日线，20-60日观察窗）
1. 上下沿各被触及≥3次且未有效突破
2. 箱体振幅：10%-25%
3. 构筑时间：≥15个交易日
4. 量能特征：箱体内缩量，上沿量能峰值低于前一波上升浪均量

#### 2.2 筹码分布强制校验
- **上方真空区**：若量度涨幅80%目标位内存在筹码密集峰（>30%），目标位下调至密集峰下沿。
- **下方锁定**：下沿±3%有无筹码密集峰？有则仓位+5%；无则仓位-5%。
- **套牢盘高压**：上方15%内筹码密集峰>40%，放弃操作。

### 第三步：突破信号与建仓

#### 3.1 有效突破三要素
- 收盘价站上箱体上沿，突破幅度 ≥ 上沿的2%
- 突破当日成交量 ≥ 箱体均量的2倍
- 突破后连续2日收盘不破上沿，或跳空+放倍量+当日不补缺

#### 3.2 建仓规则
- 次日开盘买入计划仓位的60%；回踩上沿±1.5%缩量企稳加40%。
- 仓位依据评分表：
| 评分项 | 高 | 中 | 低 |
| :--- | :--- | :--- | :--- |
| 箱体构筑 | >40交易日 | 20-40日 | 15-20日 |
| 量能倍数 | >3倍 | 2-3倍 | 刚好2倍 |
| 题材强度 | 重磅政策/业绩拐点 | 行业景气 | 一般热点 |
| 大盘配合 | 突破日大盘涨>1% | 平盘 | 下跌 |
| 筹码结构 | 下沿锁定+上无套牢 | 单项满足 | 均不满足 |
得分≥8用策略资金40%上限；5-7分用25%；<5分放弃。

### 第四步：严格止损
| 触发条件 | 动作 |
| :--- | :--- |
| 均价-5%（主板） / -7%（双创/北交所） | 无条件清仓 |
| 突破后连续2日收盘跌回箱体上沿之下 | 假突破，清仓 |
| 入场10个交易日未出现3%以上盈利 | 减半仓 |
| 突破次日开盘低开>3% | 开盘减半仓 |
| 突发利空公告（减持/立案/业绩暴雷） | 次日开盘清仓 |
| 盘中触发止损价但收盘前未收复 | 收盘清仓 |
| 累计3次止损 | 暂停30个交易日 |

### 第五步：止盈与业绩窗回避
- **第一目标位**：箱体高度80%（遇筹码峰取低者），卖出1/2，剩余止损移至成本价。
- **移动止盈**：剩余仓位3日最低价止盈。
- **业绩窗回避**：距财报披露≤5个交易日禁止开新仓；已持仓前1个交易日收盘清仓或减至1/3。

### 第六步：交易记录与归因
每笔交易记录入场/离场日期、价格、盈亏比、失败原因。

## 错误处理
- 若筹码分布数据不可得，跳过筹码校验但将仓位下调10%，并注明“筹码数据缺失，降低仓位”，突破质量评分中筹码结构项默认0分。
- 若资金池上限不足2万，返回“资金池过小，本轮放弃个股突破”。
- 若市场阶段为恐慌，直接返回“当前市场禁止个股突破交易，资金保留为现金”。

## 备注
- 本策略为短线技术博弈，与价值投资资金池物理隔离。
- 累计亏损达策略资金池20%时暂停至下季度重检。
