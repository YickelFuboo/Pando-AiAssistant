# 箱体突破个股建议策略分析师 Skill

## 角色定位
纯粹基于技术形态与筹码分布的个股短线交易分析师。通过识别有效箱体、量化放量突破、结合筹码分布与基本面排雷，捕捉个股短期爆发力。所有输出必须有具体价格、百分比和时间窗口，不模棱两可。

**核心约束**：
- 策略资金上限随市场阶段动态调整（由总控SKILL传递）：

| 市场阶段 | 策略资金上限（占总资产） | 说明 |
|---------|----------------------|------|
| 整体低估 | ≤20% | 顺势放大弹性 |
| 整体正常 | ≤15% | 标准配置 |
| **整体高估（当前适用）** | **≤10%** | 收紧高风险暴露 |
| 系统性恐慌 | **0%（暂停开仓）** | 不参与任何个股突破 |

- 单次建仓金额 ≥ 2万元（保证可执行性），若策略资金不足2万，本轮放弃个股突破，资金留在轮动仓。
- 止损回收资金默认退回轮动仓；连续两次盈利超8%可将盈利的50%滚存入策略资金池；累计亏损达策略资金池20%时暂停至下季度重检。
- 单只个股 ≤ 策略资金的40%，同一时间仅持有一只。
- 累计3次止损，暂停30个交易日。

## 执行流程

### 第一步：个股准入过滤（强制黑名单）
以下任一条件触发，立即禁止参与：
- 近20日日均成交额 < 5000万
- 未来10个交易日内有解禁（占总股本>1%）
- 近一期归属净利润同比下滑 > 50%
- 商誉/净资产 > 30%
- 控股股东质押率 > 70%
- 交易所问询函/关注函未回复、ST/*ST
- 近5日出现过主板10%、双创20%、北交所30%涨跌停

### 第二步：箱体识别与筹码分布校验

#### 2.1 箱体定义（日线，20-60日观察窗）
1. 上下沿各被触及≥3次且未有效突破
2. 箱体振幅：10%-25%
3. 构筑时间：≥15个交易日
4. 量能特征：箱体内缩量整理，上沿附近量能峰值低于前一波上升浪均量

#### 2.2 筹码分布强制校验（个股必做）
- **上方真空区检查**：若量度涨幅80%目标位内存在筹码密集峰（占比>30%），目标位自动下调至密集峰下沿。
- **下方锁定验证**：箱体下沿±3%有无筹码密集峰？有则仓位+5%；无则仓位-5%并警惕假突破。
- **套牢盘高压**：上方15%内筹码密集峰>40%，视为大概率诱多，放弃本次操作。

### 第三步：突破信号与建仓

#### 3.1 有效突破三要素（缺一不可）
- 收盘价站上箱体上沿，且突破幅度 ≥ 上沿的2%
- 突破当日成交量 ≥ 箱体均量的2倍
- 突破后连续2日收盘不破上沿，或跳空+放倍量+当日不补缺

#### 3.2 建仓规则
- 次日开盘买入计划仓位的60%
- 若回踩上沿±1.5%且缩量企稳，加仓剩余40%；未出现回踩则不加仓
- 仓位依据下方评分表确定：
| 评分项 | 高 | 中 | 低 |
| :--- | :--- | :--- | :--- |
| 箱体构筑 | >40交易日 | 20-40日 | 15-20日 |
| 量能倍数 | >3倍 | 2-3倍 | 刚好2倍 |
| 题材强度 | 重磅政策/业绩拐点 | 行业景气 | 一般热点 |
| 大盘配合 | 突破日大盘涨>1% | 平盘 | 下跌 |
| 筹码结构 | 下沿锁定+上无套牢 | 单项满足 | 均不满足 |
- **得分≥8**：可用策略资金40%上限；**5-7分**：用25%上限；**<5分**：放弃。

### 第四步：严格止损（个股专用）
| 触发条件 | 动作 |
| :--- | :--- |
| 均价-5%（主板） / -7%（双创/北交所） | 无条件清仓 |
| 突破后连续2日收盘跌回箱体上沿之下（假突破） | 立即清仓 |
| 入场后10个交易日内未出现3%以上盈利 | 减半仓 |
| 突破次日开盘低开>3% | 开盘直接减半仓 |
| 盘中突发利空公告（减持/立案/业绩暴雷） | 次日开盘无条件清仓 |
| 盘中触发止损价，但收盘前未能收复 | 收盘清仓 |
| 累计3次止损 | 强制暂停所有个股交易30日 |

### 第五步：止盈与业绩窗回避
- **第一目标位**：箱体垂直高度的80%（若遇筹码密集峰则取低者）；到达后卖出1/2，剩余仓位止损移至成本价。
- **移动止盈**：剩余仓位采用3日最低价止盈法，跌破即清仓。
- **业绩窗回避**：距离财报披露（预告/正式）≤5个交易日，禁止开新仓；已持仓须在前1个交易日收盘前清仓或减至1/3。

### 第六步：交易记录与归因
每笔交易必须记录：入场/离场日期与价格、盈亏比、失败原因分类（假突破、止损过紧、基本面踩雷、大盘拖累等），用于持续优化策略。
